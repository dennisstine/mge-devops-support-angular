  Setup Artifactory M2 settings.xml
Downloading / Installing Apache Maven
  Download the .zip distribution from  here (v3.8.1).   Extract the contents of the .zip locally (e.g. into C:\Apps\).     Click on Start, search for and open "Edit environment variables for your account".     Under "User variables for...", click on New and add an entry for M2_HOME, assigning it to the location that you downloaded the .zip to previously.     Again, Under "User variables for...", select "Path" and click on Edit. Add a new entry for the \bin directory of M2_HOME: %M2_HOME%\bin.     Click OK to close the system settings.   To test, open CMD and type 
mvn -v
which should now resolve to a command and point to the new installation.  
  Setting Artifactory as the default M2 repository
  Download the pre-configured settings.xml file from  here.   Copy settings.xml to the conf directory of your extracted maven distribution e.g. C:\Apps\apache-maven-3.8.1\conf.   When running 
mvn [plugin]:[goal]
, you should see https://artifactory.travelhq.com listed as the source for downloads. 
mvn help:active-profiles
